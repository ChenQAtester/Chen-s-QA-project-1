# My Manual QA Portfolio Project

Hi, my name is Chen. This is my manual testing project where I tested a web sign up module.

## Project Overview
I created and executed 28 test cases to check the sign-up process. I tested different scenarios like valid inputs, invalid inputs, and edge cases.

## Test Results
* Total Test Cases: 28
* Passed: 16
* Failed (Bugs): 12

## Bug Severity 
Out of all 12 bugs:
* 3 bugs are Major
* 9 bugs are Medium
* 0 bugs are Minor

The major bugs are critical because they can cause problems with the user sign up or data.

## Conclusion
Because there are 3 Major bugs and a 42.9% failure rate, my recommendation is to **not** release this version. The developers need to fix the major bugs first, and then another round of testing is required.
